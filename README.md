# مراقب حصص APRUE - برنامج استبدال المكيفات

بوت بسيط يتحقق كل 15 دقيقة من صفحة برنامج "المكيفات عالية الأداء" على
منصة APRUE (pnme.aprue.org.dz)، ويرسل إشعار تلغرام + ntfy فور فتح حصة
تسجيل جديدة.

## كيف يعمل
1. يحمّل صفحة `Clim_2000_1.php` مباشرة **بدون تسجيل دخول** — تم التأكد
   يدويًا أن الصفحة تظهر بشكل طبيعي حتى في نافذة تصفح خاصة (incognito).
2. يبحث في الصفحة عن نص "تم غلق التسجيلات".
3. إذا اختفى هذا النص ⇒ الحصة على الأرجح فُتحت ⇒ إشعار فوري.
4. يحفظ آخر حالة في `state.json` (يُحدَّث تلقائيًا داخل نفس المستودع)
   حتى لا يكرر نفس الإشعار في كل تشغيلة.
5. لو غيّر الموقع سلوكه مستقبلًا وصار يطلب تسجيل دخول فعليًا، الكود
   يتعرف على ذلك ويتجاهل تلك الدورة بدل إرسال إشعار خاطئ (راجع تعليقات
   `monitor.py`).

## خطوات الإعداد

### 1. أنشئ مستودع GitHub جديد
- اسم المستودع: اختر ما تحب (مثلاً `aprue-clim-monitor`)
- **Public** (عام) — ضروري عشان تستفيد من دقائق تشغيل غير محدودة مجانًا
- ارفع كل الملفات في هذا المشروع إليه

### 2. أضف الأسرار (Secrets)
من إعدادات المستودع: **Settings → Secrets and variables → Actions → New repository secret**

أضف الأسرار التالية (لا حاجة لأي بيانات دخول لحساب APRUE — المراقبة تتم بدون تسجيل دخول):

| الاسم | القيمة |
|---|---|
| `TELEGRAM_BOT_TOKEN` | توكن بوت تلغرام (نفس بوت أضاحي) |
| `TELEGRAM_CHAT_ID` | معرّف المحادثة (chat_id) اللي يستقبل الإشعارات |
| `NTFY_TOPIC` | `3xmgazz4bqxuiknz4ulggpta` *(اسم عشوائي وُلِّد خصيصًا لهذا المشروع — لا تشاركه)* |

### 3. اشترك في إشعارات ntfy
- ثبّت تطبيق [ntfy](https://ntfy.sh) على هاتفك
- اشترك في الـ topic: `3xmgazz4bqxuiknz4ulggpta`

### 4. فعّل الجدولة
- Actions تُفعَّل تلقائيًا بمجرد رفع ملف `.github/workflows/monitor.yml`
- **مهم:** جرّب أول تشغيل يدويًا: تبويب **Actions → APRUE Clim Quota Monitor → Run workflow**
- راجع السجلات (logs) للتأكد أن الفحص يعمل بشكل صحيح ويقرأ الصفحة فعليًا
  (لم يُختبر الاتصال الفعلي بالموقع من بيئة بناء الكود، فهذا التشغيل التجريبي مهم)

## ملاحظات مهمة
- **الجدولة كل 15 دقيقة**: GitHub Actions يعمل بتوقيت UTC وقد يتأخر التنفيذ
  بضع دقائق أحيانًا في أوقات الضغط — هذا طبيعي وليس خطأ.
- **حصانة الأسرار**: GitHub Actions يخفي تلقائيًا أي قيمة تطابق أحد الأسرار
  من كل السجلات (logs)، حتى لو المستودع عام.
- **الحفاظ على تفعيل الجدولة**: كل تشغيلة تحدّث `state.json` وتعمل commit،
  وهذا يبقي المستودع "نشطًا" فيمنع توقف GitHub التلقائي للجدولة بعد 60 يوم خمول.
- **إذا غيّرت APRUE نص رسالة الإغلاق**: افتح `monitor.py` وحدّث قيمة
  `CLOSED_MARKER` بالنص الجديد.
- هذا البوت **للإشعار فقط** ولا يقوم بالتسجيل التلقائي في البرنامج —
  عليك التسجيل يدويًا بمجرد وصول الإشعار.
